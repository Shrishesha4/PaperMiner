# **App Name**: InsightMiner

## Core Features:

- CSV Upload: Accept a CSV file downloaded from IEEE Xplore containing research paper metadata.
- Title Categorization: Employ a generative AI tool to categorize research paper titles based on the content of the titles.
- Statistical Visualization: Display statistical graphs visualizing the distribution of research papers across different categories.
- Keyword display: Show a list of the most frequent title keywords to help give insight into the content of the archive. Implement a way to rank/weight these keywords
- Statistical Parameters: Implement parameters such as 'Publication Year', 'Author Keywords', and 'IEEE Terms' to show statistical parameters for papers based on user selection.
- Dynamic Filtering: Enable dynamic filtering of research papers based on category to allow user refinement.

## Style Guidelines:

- Primary color: Dark slate blue (#3B5998) to reflect the serious nature of the research topics being examined, with an echo of IEEE's brand identity.
- Background color: Light gray (#F0F2F5) to provide a clean, neutral backdrop that does not distract from the information being presented.
- Accent color: Soft violet (#8075FF) to highlight interactive elements and key data points, while maintaining visual harmony.
- Body and headline font: 'Inter' (sans-serif) for clear and accessible information display, with emphasis on readability for technical content.
- Use minimalistic icons to represent categories and filter options.
- Employ a grid-based layout to ensure structured presentation of statistical graphs and filtered data.
- Apply subtle transitions when filtering and displaying data for a smooth user experience.
